User Guide Generate Low Fidelity to High Fidelity

  • Open website L2H
  • Head to Try Converting or click `Try Now`
  • Input SALT UML code
  • Click `Convert` button
  • The system processes saltUML code from user, and display interface high fidelity HTML with CSS in two version (2 template)
  • User can choose template
  • The system provides a template code based on user choice
  • User can copy HTML and CSS code based on selected high fidelity template, with click icon `copy`
